🌌 Quantum Computing Laboratory Portal
An elite, high-performance web platform designed for the Center for Quantum Communication and Computing at SRMIST-KTR. Engineered with a premium Gold Glassmorphic layout system, this terminal showcase serves as a world-class portal tracking next-generation computational paradigms, localized algorithmic frameworks, secure cryptographic communication hardware, and active institutional MoUs.

🎨 Design Language & Philosophy
The user interface breaks away from traditional dashboard frameworks, adopting a unified deep-space obsidian environment accented by metallic gold micro-fuses and localized glass panels.

Obsidian Canvas: #030302 base layer preventing bleed and keeping ambient focus high.

Gold Glassmorphism: Dense dark glass panels (bg-[#0a0904]/70) bound by subtle gold vectors (border-amber-900/30).

Unified Fluid Transitions: Asymmetric layout segments and smooth component mount routines powered by Tailwind hardware acceleration.

⚡ Key Features & Structural Architecture
Central Lab Terminal Engine (Home.jsx): Seamless landing array presenting the core branding identity alongside a localized, responsive laboratory metrics dashboard.

Advanced Equipment Portfolio (Equipments.jsx): Alternating glass-plate display showcasing operational physical kits (QRNG entropy harvesters, Fiber QKD benches, Topological wave modulators) and upcoming hardware pipelines.

Curriculum & Electives Console (Subjects.jsx): Interactive accordion engine mapping both foundational core algorithms and advanced elective paths with fluid, rotatable geometric assets.

Strategic MoU Tracking Display (MoUs.jsx): Automated partner gallery utilizing adaptive filters (brightness-0 invert opacity-40) to smoothly render corporate brand layouts.

Industrial Visits Track (IndustrialVisits.jsx): Structured repository displaying elite field visits (IBM, ISRO, Google Quantum AI) with unified silver-to-chroma transition cards.

🛠️ Tech Stack & Dependencies
The system is constructed completely on decoupled modern front-end layers:

Core Framework: React 18+ (Functional components, Hooks layout paradigm)

Styling Engine: Tailwind CSS v3+ (Utility-first, customized arbitrary configurations)

Routing System: React Router DOM v6 (Secure client-side navigation paths)

Visual Enhancements: Native CSS Backdrop-Blurs & PostCSS optimization matrices
